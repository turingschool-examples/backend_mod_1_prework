# Boolean Exercises

1. true && true --> **true**
1. false && true --> **false**
1. 1 == 1 && 2 == 1 --> **false**
1. "test" == "test" --> **true**
1. 1 == 1 || 2 != 1 --> **true**
1. true && 1 == 1 --> **true**
1. true || 1 == 1 --> **true**
1. "test" == "testing" --> **false**
1. 1 != 0 && 2 == 1 --> **false**
1. "test" != "testing" --> **true**
1. "test" == 1 --> **false**
1. !(true && false) --> **true**
1. !(1 == 1 && 0 != 1) --> **false**
1. !(10 == 1 || 1000 == 1000) --> **false**
1. !(1 != 10 || 3 == 4) --> **false**
1. !("testing" == "testing" && "Zed" == "Cool Guy") --> **true**
1. 1 == 1 && (!("testing" == 1 || 1 == 0)) --> **true**
1. "chunky" == "bacon" && (!(3 == 4 || 3 == 3)) --> **false**
1. 3 == 3 && (!("testing" == "testing" || "Ruby" == "Fun")) --> **false**

## Equality Operators

* == Equal
* != Not equal
* > Greater than
* < Less than
* >= Greater than or equal to
* <= Less than or equal to
* <=> Combined comparison
