### Ruby Equality operators
  * == : Equal
  * != : Not Equal
  * > : Greater Than
  * < : Less Than
  * >= : Greater than or equal to
  * <= : Less than or equal to
  * <=> : Combined comparison operator.
  * === : Test Equality
  * .eql? : True if two values are equal and same type
  * equal? : True if two things are same object
