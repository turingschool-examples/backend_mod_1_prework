a process on given arguments.

1. Create a method named `hello` that will print `"Sam I am"`.
```

```

1. Create a method named `hello_someone` that takes an argument of `name` and prints `"#{name} I am"`.
```
def hello_someone(name)
  puts "#{name} I am"
end
```
